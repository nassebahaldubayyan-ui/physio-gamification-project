# Database Files for Rehabilitation System

## 📁 Files Overview

| File Name | Purpose | When to Use |
|-----------|---------|-------------|
| `init.sql` | Complete database with all tables and sample data | First time setup |
| `alter_tables.sql` | Only the new columns and tables | Adding features to existing DB |
| `sample_data.sql` | Just the sample progress data | Testing charts and graphs |
| `backup_complete.sql` | Full database backup | Restoring from backup |
